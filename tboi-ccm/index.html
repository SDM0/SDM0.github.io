TBOI Custom Challenge Maker

By SDM_0

List of collectible IDs: https://platinumgod.co.uk/

Wiki of custom challenge infos: https://bindingofisaacrebirth.fandom.com/wiki/Custom_Challenges

IsaacGuru's video on custom challenges (very useful to find the directory and setup files): https://www.youtube.com/watch?v=8BVOEu6kYWQ

Important notes:

-You can leave "insert" and "select" options empty/unselected

-Press Ctrl (Windows) or Command (Mac) key while selecting multiple options in "select" (no dragging needed)

-Trinkets: You need to add an increasing trinket slot item if you want 2 trinkets (Mom's Purse(139) or Belly Button(458))

-Hearts: Even number = full heart / Odd number = half heart
